5. Frammistaða, Git er kraftmikið og öruggt kerfi í samanburði við önnur Version Control forrit. Öryggi, Git er hannað til að sérstaklega viðhalda kóða, allt sem er gert í Git er búið að tryggja öryggi með dulkóða. Sveigjanleiki, Git er þægilegt út af það styður við allskonar þróunarverkflæði og allskonar verkefni hvort sem þau eru lítil eða stór.
